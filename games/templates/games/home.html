Sobre o Projeto

Objetivo

Este é um Progressive Web App (PWA) desenvolvido como Trabalho de Desenvolvimento de Engenharia (TDE). O objetivo é demonstrar a criação de uma aplicação web moderna que permite aos usuários explorar e jogar jogos retro através de um emulador embutido.

Tecnologias

  • Backend: Django (Python)
  • Frontend: Bootstrap 5, HTML5, CSS3
  • PWA: Service Worker, Web App Manifest
  • Emulador: retrogames.cc (via iframe)
